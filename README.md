# Color-Trigger-Bot
Advanced Universal Color Trigger Bot For Ahk (it doesnt works with version 2)
# How To Use (U Can Change Keybinds)
2 Color Mode Supported
Auto Scan = Auto Scans Selected Color And Shoots For You
Left Click Macro (Rapid Fire) = Its just normal m1 macro
# How Do I Set Colors ?
There Is A Color1 And Color2 In User Settings Section :
color_1        :=     0xFFF200 
color_2        :=     0xA145A3
u can change them however you want
# What Is Color Sensibility ?
It is basically a setting for color detection the higher you put the more accurate the correct color will be detected but it might delay your shot so dont overdo it
